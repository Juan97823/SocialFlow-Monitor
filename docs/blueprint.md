# **App Name**: DevStock SAS Monitor

## Core Features:

- Dynamic Analytics Dashboard: Interactive data visualization center using Chart.js to monitor hourly activity, active users, and system performance.
- Distributed Task Orchestrator: Functional module to create, schedule, and track tasks through states like Pending, In Progress, and Completed.
- AI Sentiment Simulation Tool: Generative tool to analyze and categorize simulated interaction data into positive, neutral, or negative sentiment buckets.
- Real-time Observability Stream: Live feed panel displaying system logs, response times, and alert notifications for proactive monitoring.
- MySQL Data Persistence: Structured storage for users, roles, audit trails, and task histories using relational database principles.
- Administrative Control Panel: A comprehensive management interface for User CRUD, role assignments, and full system configuration settings.
- Secure Access Management: Unified authentication system with session management, password recovery, and frontend/backend validation.

## Style Guidelines:

- Primary color: Electric Blue (#3399FF), representing the pulse of digital interaction against a high-tech backdrop.
- Background color: Deep Onyx (#0F1216), a heavily desaturated blue hue providing a sophisticated dark mode environment.
- Accent color: Neon Cyan (#1AE6E6), used for active states, live indicators, and key system alerts to ensure maximum contrast.
- Font pairing: 'Space Grotesk' for tech-oriented, futuristic headlines and 'Inter' for clear, machined UI text and readable data metrics.
- Code font: 'Source Code Pro' used specifically for real-time observability logs and administrative audit entries.
- Linear, minimalist icons with thin strokes and subtle glows that reflect a modern SaaS dashboard aesthetic.
- Smooth transitions for the collapsible sidebar and CSS-driven 'pulse' effects on live data points and system status badges.
